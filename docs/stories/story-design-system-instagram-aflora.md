# Story: Design system baseado no Instagram da Aflora

## Objetivo

Criar um design system consistente usando as cores e o estilo vistos no Instagram da marca.

## Tarefas

- [x] Extrair e consolidar paleta principal da marca.
- [x] Definir tokens semanticos (marca, neutros e estados).
- [x] Definir tipografia, espacamento, raio e sombras.
- [x] Documentar regras de uso (proporcao, acessibilidade e aplicacao).
- [x] Gerar arquivo de tokens CSS reutilizavel.

## Criterios de aceitacao

- [x] Paleta contem lilas, lima, pink, verde e neutros usados no perfil.
- [x] Existe arquivo CSS com variaveis (`:root`) pronto para uso.
- [x] Existe guia em markdown com orientacoes de aplicacao.
